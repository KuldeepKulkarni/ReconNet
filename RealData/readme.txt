This folder contains real data obtained from a block compressive imager at the Intelligent Imaging and Sensing Lab at University of Arizona. 

The imaging hardware is described in detail in the following paper:

Ronan Kerviche, Nan Zhu, Amit Ashok, "Information Optimal Scalable Compressive Imaging System", Classical Optics, 2014, Optical Society of America, 2014

If you use this data, please cite the following papers

1. Kuldeep Kulkarni, Suhas Lohit, Pavan Turaga, Ronan Kerviche, Amit Ashok, "ReconNet: Non-Iterative Reconstruction of Images From Compressively Sensed Measurements", [The IEEE Conference on Computer Vision and Pattern Recognition (CVPR), 2016] (http://cvpr2016.thecvf.com/), pp. 449-458.

2. Suhas Lohit, Kuldeep Kulkarni, Ronan Kerviche, Pavan Turaga, Amit Ashok, "Convolutional Neural Networks for Non-Iterative Reconstruction of Compressively Sensed Images", Transactions on Computational Imaging, 2018
